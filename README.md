# ArpiDent App

Учебное веб-приложение для онлайн-записи пациентов в стоматологическую клинику ArpiDent.

## Что реализовано

- frontend сайта клиники;
- backend на Node.js;
- файловая база данных `db.json`;
- REST API для врачей, услуг, записей, отзывов и админ-панели;
- онлайн-запись пациента;
- проверка занятости временного окна;
- сохранение записей в базе данных;
- личный кабинет пациента;
- панель администратора;
- отзывы пациентов;
- простая учебная авторизация администратора.

Проект специально сделан без внешних npm-зависимостей, чтобы его можно было запустить максимально просто.

## Запуск проекта

1. Установить Node.js.
2. Открыть папку проекта в терминале.
3. Запустить приложение:

```bash
npm start
```

или напрямую:

```bash
node server.js
```

4. Открыть в браузере:

```text
http://localhost:3000
```

## Доступ в админ-панель

На сайте открыть раздел **Админ**.

```text
Логин: admin
Пароль: admin123
```

## Принцип работы

Пациент выбирает услугу, врача, дату и время. Frontend отправляет данные на backend через `fetch`. Сервер проверяет корректность данных и занятость слота. Если окно свободно, запись сохраняется в `db.json`. После этого запись появляется в личном кабинете пациента и в панели администратора.

## Важно

Проект является учебным. Данные пациентов, врачей, отзывов и записей используются в демонстрационных целях.

## Как проверить, что это не просто frontend

1. Откройте `http://localhost:3000/api/health` — если появляется JSON-ответ, значит работает backend.
2. Создайте запись на сайте, затем откройте раздел «Админ» — запись появится там из базы `db.json`.
3. Откройте файл `db.json` после создания записи — новая запись будет сохранена в разделе `bookings`.
4. Если остановить сервер в терминале, сайт по адресу `http://localhost:3000` перестанет открываться. Это значит, что проект работает через серверную часть.
